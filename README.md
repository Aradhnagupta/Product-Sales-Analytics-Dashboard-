# Product Sales Analytics Dashboard

## Project Overview

This project focuses on analyzing product sales data to generate meaningful business insights using Microsoft SQL Server and Power BI. The objective is to clean and transform product data, calculate key business KPIs, and build an interactive dashboard that enables stakeholders to monitor product performance and make data-driven decisions.

The project follows a complete analytics workflow, starting with data preparation in SQL Server, followed by KPI development using DAX in Power BI, dashboard creation in a test environment, and final deployment to production.

---

## Project Objectives

* Clean and transform product sales data using Microsoft SQL Server.
* Develop key business KPIs and interactive visualizations using Power BI.
* Deploy the completed dashboard from the test environment to production.
